# Part 3: The Spider's Web

> **Adventure**: Lost Mines of Phandelver | **Pages**: 27–41 | **Levels**: 3 → 4

## Overview

A sandbox section where the party follows up on leads from Phandalin. Characters are **not required** to visit all locations — they pick quests based on which NPCs they met and which hooks they picked up. All paths eventually converge on **Wave Echo Cave** (Part 4).

> [!info] Sandbox Structure — Quests Interlock
> This is a player-driven sandbox. Agatha ↔ Old Owl Well ↔ Wyvern Tor form a quest triangle. Thundertree's Reidoth gates access to Wave Echo Cave behind the Venomfang encounter. Cragmaw Castle is the main-quest path. Let the party choose their route — all roads lead to Part 4.

### Cross-Location Dependencies

```
Agatha's Lair ←→ Old Owl Well
  Kost wants Agatha's answer (Arthindol)
  Agatha can answer Kost's question

Old Owl Well → Wyvern Tor
  Kost wants the orcs removed

Wyvern Tor → Old Owl Well
  Completing this fulfills Kost's request

ALL PATHS → Wave Echo Cave (Part 4)
```

> [!tip] Pacing — Don't Run Every Location
> The party doesn't need to visit all five locations. Two or three is typical. If they're eager to rescue Gundren, Cragmaw Castle (~1 day travel) is the fastest path to Part 4. If they want to level up first, Thundertree and the side quests provide enough XP to reach level 4.
