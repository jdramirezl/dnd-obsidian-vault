# Monster Manual — Bestiary

**Source**: Monster Manual (5th Edition, 2014) — 354 pages

A complete breakdown of every creature in the Monster Manual into individual Obsidian-linked files.

## Structure

### [[00-Introduction|Introduction]]
How to read stat blocks, creature types, size categories, CR/XP tables.

### [[By-Name/A/Aboleth|By-Name/]] — 187 files
One file per monster (or monster group), organized alphabetically A–Z. Each file contains:
- Full stat block with markdown ability score tables
- Lore/flavor text
- DM Quick Reference with tactical advice and ```````[[wikilinks]]``````` to conditions

### [[By-Type/README|By-Type/]] — 14 type indexes
Aberrations, Beasts, Celestials, Constructs, Dragons, Elementals, Fey, Fiends, Giants, Humanoids, Monstrosities, Oozes, Plants, Undead.

### [[By-CR/README|By-CR/]] — 29 CR level indexes
CR 0 through CR 30. Every stat block indexed at its correct CR.

### Appendices/ — 2 files
- [[Appendix-A-Miscellaneous-Creatures|Appendix A]]: 95 beasts and simple creatures (summary table + compact stat blocks)
- [[Appendix-B-NPCs|Appendix B]]: 21 NPC stat blocks (Acolyte through Veteran)
