# Drow Tactics: Arachnomancers

> [Source](https://www.themonstersknow.com/drow-tactics-arachnomancers/)
> Bestiary: [[Elves-Drow]]

---

## Tactical Summary

- Drow warlock; shapeshifts into giant spider (bonus action); casts all spells at 5th level, limited to 3 leveled spells per encounter + innate spells
- Long-range spellslinger (extraordinary INT, very high DEX/CHA); Poisonous Touch and Bite are self-defense
- Has blindsight, Spider Climb, Web Walker in both forms; prioritizes killing enemy spellcasters

## Combat Behavior

- **Game plans** (pick by situation):
  1. *Single fragile caster*: *hold monster* → rush in with Poisonous Touch/Bite
  2. *Multiple casters*: *web* them → all allies get advantage vs restrained
  3. *With allied mages*: allies *web* → *insect plague* on restrained enemies → cantrips at range
  4. *With allied webbers*: allies web → *conjure animals* (16 giant wolf spiders) → mob with advantage → cantrips
  5. *Non-casters in confined space*: *fear* → allies melee → cantrip potshots
- *Chill touch* (3d8, suppresses healing) for assassination; *eldritch blast* (3 bolts) for targets of opportunity
- *Darkness* only if it covers arachnomancer + enemies but NOT allies
- Spider form web attack (no concentration) sets up combos with allies

## Key Decision Points

- Shape choice: spider form's main advantage is non-concentration web spinning
- *Dispel magic* only for effects causing very serious trouble (opportunity cost)
- Skip variant Summon Demon (50% failure too risky) — script the shadow demon in if you want it
- Let other drow handle *faerie fire*; don't waste actions on *dancing lights* or *levitate*

## Retreat Conditions

- Zealot but intelligent — retreats if objective is unachievable, not from damage alone
- Uses *dimension door* or *etherealness* to escape; abandons allies without hesitation
- If escorting a VIP, favors *dimension door* (brings one passenger)

> [!tip] The humble *web* may be the arachnomancer's best concentration spell — restrains targets (advantage on all attacks) and the arachnomancer ignores it entirely via Web Walker + Spider Climb.
