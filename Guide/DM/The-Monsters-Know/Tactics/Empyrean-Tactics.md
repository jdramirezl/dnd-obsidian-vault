# Empyrean Tactics

> [!summary]
> Demigod-tier boss (CR 23). Str/Con 30 brute with AC 22 unarmored, 313 hp, immunity to nonmagical physical damage, advantage on saves vs magic. Casts *earthquake* immediately, charges and bashes, *fire storm* when moderately wounded, *plane shifts* when seriously hurt. Tier 4 encounter.

---

## Stat Block Profile
- **Type:** Celestial, Huge, chaotic good (or evil if corrupted)
- STR 30, CON 30, Int 21 — brute to rule all brutes
- AC 22 (unarmored), HP 313, Speed 50 ft / fly 50 ft
- Immune to nonmagical physical damage; advantage on saves vs magic
- Legendary Resistance ×3/day; maul can instantly kill PCs with ≤23 hp
- Bolt: 600-ft range, auto-selects optimal damage type (Int 21)

---

## Combat Sequence

### Round 1: Earthquake
- Cast immediately — sustained for entire combat
- Creates difficult terrain (empyrean flies over it), disrupts concentration, knocks prone, opens fissures
- Hovers 1–2 ft off ground throughout

### Default: Charge and Bash
- Close to melee → maul attack; Bolt (600 ft) only vs unreachable enemies
- Int 21 → knows which damage type hurts each target most

### Moderately Wounded (≤219 hp): Fire Storm
- 1/day, 7d10 fire damage engulfing all opponents
- Cast when it takes opponents seriously — not opener, not last resort

### Legendary Resistance
- Uses every time it fails a save — doesn't bank uses
- Weakest save: Dex

---

## Legendary Actions (3/round)
- **Attack (1):** additional maul or Bolt — always the default (compensates for no Multiattack)
- **Bolster (1):** only when empyrean has allies
- **Trembling Strike (2):** 60-ft radius prone — crowd control. Use only if all PCs in range, or 12+ enemies total

---

## Retreat

### Plane Shift (≤125 hp)
- Escape hatch when seriously wounded; PCs at this tier may pursue

### If Plane Shift Used
- 1–2 melee opponents: Dodge while retreating
- 3+ melee opponents: Disengage while retreating
- Dash only if enemies can match 50 ft but not 100 ft (Int 21 — it would know)
- Continues attacking with legendary Bolt while retreating
- Trembling Strike if chased by 12+ enemies

> [!tip] DM Tips
> *Earthquake* round 1, every time. Default legendary action is always Attack. *Fire storm* is the "taking you seriously" moment. Corrupted empyreans as exiled material-plane tyrants make great BBEGs.
