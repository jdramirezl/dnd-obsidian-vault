# The Monsters Know What They're Doing

> **Source**: [themonstersknow.com](https://www.themonstersknow.com/) by Keith Ammann
> **Total Files**: 316 tactics analyses

Condensed tactical analyses of how to run D&D monsters intelligently, based on Keith Ammann's blog and books. Each file links to the original article and provides actionable DM advice.

## How to Use

Each tactics file covers:
- **Tactical Summary** — Core behavior in 3-5 bullets
- **Combat Behavior** — Opening moves, preferred tactics, group coordination
- **Key Decision Points** — Critical choices during combat
- **Retreat Conditions** — When and how the monster flees
- **DM Tip** — One practical callout

## Cross-References

Most files link to the corresponding [[00-Introduction|Bestiary]] stat block. Look for the `Related` line at the top of each file.

## Browse

All files are in the [[Tactics/]] subdirectory, named by monster (e.g., `Goblin-Tactics.md`, `Beholder-Tactics.md`).

Use Obsidian search or tags (`#tactics`, `#monsters-know`) to find specific monsters.
