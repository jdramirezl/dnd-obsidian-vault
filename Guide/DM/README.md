# DM Resources

Index of all Dungeon Master reference material.

## Resources

### [[return-of-the-lazy-dungeon-master/README|Return of the Lazy Dungeon Master]]
> **Files**: 34 | **Source**: Sly Flourish

A streamlined session-prep method. Covers the eight steps of lazy prep, running your game at the table, and thinking about your campaign long-term. The go-to framework for efficient DM preparation.

### [[The-Monsters-Know/README|The Monsters Know What They're Doing]]
> **Files**: 316 | **Source**: Keith Ammann

Tactical analyses for running D&D monsters intelligently. Each file covers a monster's combat behavior, preferred tactics, retreat conditions, and group coordination. Look up any monster before you run it.

### [[Angry-GM/README|The Angry GM]]
> **Files**: ~490 | **Source**: The Angry GM

Condensed analyses of articles covering game design, encounter building, adventure design, narration, and core GM philosophy. Deep dives into *why* RPG mechanics work the way they do and how to use them better.
