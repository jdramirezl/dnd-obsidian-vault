# True Mechanical Managery — Introduction

**Source:** [True Mechanical Managery – The Angry GM](https://theangrygm.com/true-mechanical-managery/)
**Date:** September 4, 2024
**Series:** True Campaign Managery — True Mechanical Managery Module

---

## Summary

This article introduces the True Mechanical Managery module — a sub-series within True Campaign Managery focused on how GMs should deliberately manage their game system's rules beyond just knowing what the rules say. The rulebooks tell you *what* happens mechanically, but they don't tell you *how* to administer those mechanics at the table, *when* to apply them, or *why* the practical implementation matters. True Campaign Managers make thoughtful, deliberate, forward-looking choices about all of this before the first session.

---

## Key Concepts

### Mechanical Management Defined
- The rules tell you the mechanical code (XP values, initiative, death saves, etc.) but not the practical procedures for administering them at the table.
- **Mechanical Management** = making good, logical, deliberate choices about how you plan to use the game rules and what rules you'll use to give your campaign the best chance to succeed.
- It's about implementation, not design. You're not rewriting the game — you're deciding how to run it.

### Why It Matters
- **Pacing:** When do you award XP? When do players level up? Stopping mid-session for bookkeeping kills momentum.
- **Balance:** Character death removes a party member's resources and abilities. Replacement characters create balance questions.
- **Plot:** A dead character can sever an entire plotline. New characters need to be woven in.
- **Player experience:** A player without a character is a player not playing. That's urgent.
- **Vision support:** The right mechanical choices amplify your campaign vision. The wrong ones undermine it.

### Three Impacts of Mechanical Choices
1. **Support the game you're running** — Good choices emphasize themes, setting, and tone. Example: Honor/Renown rules for a game about personal glory; dropping rogues and necromancers from an honor-focused campaign.
2. **Avoid weighing it down** — Wrong choices drag the game. Example: Stopping after every encounter for XP math in a game with inconsistent scheduling wastes precious table time.
3. **Solve problems before they're problems** — Character death is predictable. XP thresholds are predictable. New players joining is predictable. Have a plan for all of it.

### Optional and Variant Rules
- Every system has tons of optional rules scattered through the books. Many GMs don't even realize which rules they're using are optional (Point-Buy, Feats, and grid combat are all variants in D&D 5E).
- True Campaign Managers sift through all options deliberately. Mere Campaign Supervisors operate on assumptions and do whatever everyone else does.
- Splatbooks (supplemental rulebooks) pile on even more options. Each changes the game somehow — themes, pacing, complexity, or available choices.

### Character Creation — The Overloaded Problem
- Character creation is where option overload hits hardest. Too many choices, too little context for new players, too much time spent not playing.
- GMs must know what every character can do and how it works. Every unresolvable option is a drag on the campaign.
- Most GMs barely think about the character creation process — they just turn players loose. True Campaign Managers handle it deliberately.

---

## Actionable Takeaways

- Before the campaign starts, review every mechanical subsystem you'll use and decide how you'll administer it at the table.
- Identify which optional/variant rules you're using and make those choices consciously, not by default.
- Have a plan for: XP and advancement timing, character death, replacement characters, and new players joining.
- Match your mechanical choices to your campaign vision — use them to reinforce themes and structure.
- Don't wait to be blindsided by predictable problems. If the rules include character death, plan for it now.
