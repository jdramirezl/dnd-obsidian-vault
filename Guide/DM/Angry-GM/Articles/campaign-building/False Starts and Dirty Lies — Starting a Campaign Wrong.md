# False Starts and Dirty Lies — Starting a Campaign Wrong

**Source:** [False Starts and Dirty Lies: Starting a Campaign Wrong – The Angry GM](https://theangrygm.com/false-starts-and-dirty-lies-starting-a-campaign-wrong/)
**Date:** December 18, 2017
**Series:** Start Your Own Campaign

---

## Summary

A "False Start" is a campaign structure where the first act feels like a different campaign before the real campaign begins. It's a powerful but dangerous tool — it can bore players with busywork before the real hook, or betray them by yanking away a game they were enjoying. Used correctly, it makes a great campaign even better by establishing emotional connections, teaching mechanics, or layering mystery before the main plot arrives.

---

## Key Concepts

### What a False Start Is
- The first act of the campaign establishes a different tone, scope, or premise than the real campaign.
- At some point, a revelation or event shifts the campaign into its true form.
- It's one of many campaign structure features — not a classification system. Think of it like a couch that also turns into a bed: it's a feature you can combine with other structural choices.

### Why False Starts Are Terrible
They risk two failure modes:
1. **Boring the players** — burying the exciting campaign behind hours of pointless busywork.
2. **Betraying the players** — drawing them into something they enjoy, then pulling the rug out.

The start of any campaign is both a **sampler** (this is what the game feels like) and a **promise** (this is what you'll be doing). A False Start breaks one or both.

GMs are seduced by the **inciting moment** — the dramatic reveal when everything changes. But that moment is usually just as powerful (or more so) as the campaign's actual opening. You don't need weeks of buildup to make it land.

### Why False Starts Can Be Great
They work when the False Start **demonstrably improves the real campaign** in ways you couldn't achieve otherwise.

**Chrono Trigger** as the exemplar:
- The early hours teach the player how time travel works — its power and its dangers — so that when the apocalypse is revealed, the player sees a path forward instead of an impossible task.
- The early hours build emotional connections to companion characters, so the player cares about saving the world through those characters' reactions.

### Valid Reasons to Use a False Start

1. **Characters and Connections** — Give players time to build personal goals, relationships, and emotional investment before a massive A-plot overwhelms everything. Only valid when the real campaign is so huge it would push all personal stuff to the side.

2. **Establishing B-Plots Before the A-Plot** — Set up minor plot threads that will run alongside the main campaign. Without the False Start, these threads would feel like silly distractions once the main plot arrives.

3. **Layers of the Onion** — The campaign is structured around peeling back layers of mystery. The False Start is the surface layer — the characters walk around on top of a mystery they don't know exists. The real campaign begins with the first revelation.

### Executing a Good False Start
Three requirements:

1. **Know exactly what the False Start is supposed to do** — and design it to actually accomplish that. It's a real campaign with its own shape and plot threads. Don't assume it'll just "happen."

2. **Glue** — You need something holding the party together during the False Start AND after the real campaign begins. They're two different campaigns; they both need party cohesion. The False Start can actually solve this: organically-built relationships during the False Start can become the glue for the real campaign.

3. **The False Start must be fun** — Almost as fun as the real campaign. If the real campaign is a 10, the False Start needs to be a 9.5. If it's boring, players will be fatigued or checked out by the time the real campaign starts.

### The Secret
> "I have never run a False Start campaign without telling the players that it was a False Start campaign right from the get-go."

Don't surprise your players with the shift. Tell them upfront. The False Start is a structural tool, not a gotcha.
