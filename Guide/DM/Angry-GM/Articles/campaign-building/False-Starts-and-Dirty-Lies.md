## Summary

A **False Start** is a campaign structure where the first act feels like a different campaign from the one you actually intend to run. The "real" campaign begins after a pivotal revelation or shift. Angry uses *Chrono Trigger* as the exemplar: the game opens with small-scale time-travel mishaps before revealing the true stakes — preventing an eldritch apocalypse.

The default advice is correct: **start as you mean to go on.** The opening of a campaign does two things — it establishes what the campaign is about (the promise) and it sets the tone for how it will feel (the sampler). A False Start deliberately breaks both of those rules, which is why it's usually terrible.

False Starts fail in two ways: they **bore** players with busywork before the real campaign, or they **betray** players by pulling the rug out from under something they were enjoying. GMs are seduced by the "big reveal moment" but that moment is almost always just as powerful — or more powerful — as a cold open.

However, False Starts *can* make great games better when they serve a specific, articulable purpose: building emotional connections, establishing B-plots before the A-plot overwhelms them, or layering an onion-style mystery. The key is that the False Start must *improve* the real campaign in a way that couldn't be achieved otherwise.

**Critical secret:** Angry has never run a False Start without telling the players upfront that it was a False Start.

## Key Advice

- **Default rule: start your campaign when it starts.** Don't start a different campaign and then switch. The opening sets the promise and the tone.
- **False Starts are usually terrible.** They either bore players with irrelevant busywork or betray them by yanking away something they liked.
- **The "big moment" isn't worth it alone.** GMs overestimate how much better a reveal is after being strung along. If the reveal is cool, it's cool as a cold open too.
- **Only use a False Start if it demonstrably improves the game.** You must be able to state clearly what the False Start does that couldn't be done any other way.
- **Valid reasons include:**
  - Building character relationships and emotional connections before an overwhelming main plot takes over.
  - Establishing B-plots and minor threads before the A-plot dominates everything.
  - "Layers of the Onion" — the False Start *is* the campaign's surface layer, and the real start is the first revelation.
- **A False Start is two campaigns.** Both need structure, glue (what holds the party together), and must be fun to play. The False Start must hit ~95% of the fun level of the real campaign.
- **The False Start needs its own glue.** Don't assume the real campaign's motivation will retroactively hold the False Start together.
- **Tell your players.** Never surprise them with a False Start. Transparency prevents the boredom and betrayal problems.

## Actionable Steps

1. **Ask yourself: does my campaign need a False Start?** If you can't articulate a specific benefit beyond "the reveal will be cool," don't use one. Just start the real campaign.
2. **If yes, define the purpose.** Write down exactly what the False Start accomplishes: relationship building, B-plot establishment, onion-layer mystery, etc.
3. **Design the False Start as its own mini-campaign.** Give it structure, a shape, plot threads, and its own party glue. It must stand on its own as a fun experience.
4. **Ensure the transition works.** Plan what keeps the party together *after* the shift. Consider whether the False Start can organically build relationships strong enough to serve as glue later.
5. **Make the False Start almost as fun as the real campaign.** If the real campaign is a 10/10, the False Start needs to be at least 9.5/10.
6. **Tell your players it's a False Start.** Be transparent about the structure. This prevents boredom and betrayal.
7. **Test the transition moment.** Ask: will players who enjoyed the False Start be excited about the shift, or will they feel cheated? If cheated, redesign.

> [!tip] The Chrono Trigger Test
> Before committing to a False Start, ask: "Does my opening act teach the players something essential about the tools, world, or relationships they'll need for the real campaign?" If the answer is no — if it's just delay before the cool part — skip it and open with the cool part.
