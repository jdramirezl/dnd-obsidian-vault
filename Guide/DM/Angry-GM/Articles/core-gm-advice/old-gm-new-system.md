# How to Teach An Old GM a New System

**Source:** [The Angry GM](https://theangrygm.com/old-gm-new-system/)
**Date:** November 29, 2023
**Category:** Core GM Advice

---

## Summary

A guide for experienced GMs on how to properly learn a new TTRPG system. Angry argues that the worst ways to learn are: reading the rulebook cover-to-cover first, watching YouTube actual plays, or just jumping in and running it yourself with your existing GM habits. Instead, the best approach is to use a Starter Set (or build your own equivalent) with pre-generated characters and a published adventure, run a throwaway tutorial campaign of 3–5 sessions, and read the full rulebook concurrently while running the game.

---

## Key Points

### The Three Worst Ways to Learn a New System

1. **The Rulebook Alone** — RPG rulebooks are terrible teaching tools. You'll eventually need to read the whole thing, but don't start there.
2. **YouTube / Watching Others Play** — You can't verify they're playing correctly, you only see the player-facing side, and you're probably not giving it full attention anyway.
3. **Running It Yourself (as an Experienced GM)** — Experienced GMs skim rules, assume systems are interchangeable, immediately start tinkering, and end up making every system feel the same. Your ego gets in the way.

### The Right Approach: Let the System Speak for Itself

- **Use a Starter Set** — Most publishers offer one. It contains abbreviated rules, a module, and pre-generated characters. The best ones teach rules gradually through the adventure itself.
- **Commit to a throwaway tutorial game** — Run 3–5 sessions of a published adventure with pregens. You're not emotionally invested; you know it'll suck. Throw it all away when done.
- **If no Starter Set exists:**
  - Look for Free RPG Day materials, crowdfunding previews, or convention packs.
  - DIY: grab a published module from the game's creators, build pregens yourself, and find abbreviated rules if available.
  - **Last resort:** Read the adventure module first, look up rules in the core book only as you encounter them in the module, and learn just enough to muddle through session one.

### While Running the Tutorial Game

- **Read the full rulebook** from cover to cover concurrently. Don't try to learn from it — just read it. The combination of practice, immersion, and context makes retention much easier.
- You can skim fiddly character options, but read every rule, every race/class description, every skill — even corner cases like drowning and grappling. Just knowing a rule exists somewhere is valuable.

### After the Tutorial

- You're ready to start a real campaign with bespoke characters and your own adventures.
- **Don't tinker with the rules yet.** Run the game as the publisher intended for at least 12 sessions before modifying anything.
- The publisher is still smarter than you about their own system until you've logged enough hours.

---

## Core Principle

> Approach every new system as if you're a complete beginner. Assume you know nothing. Start from scratch. Your experience with other systems is as likely to mislead you as help you.
