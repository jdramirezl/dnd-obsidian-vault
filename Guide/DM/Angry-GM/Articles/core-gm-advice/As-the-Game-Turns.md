## Key Advice

- **Every action resolves the same way regardless of scale.** Searching a room and searching a forest are mechanically identical: one decision, one resolution, one narration. The only difference is how much game time passes.
- **The Arbitrary Game Turn has three steps:** (1) Evaluate the action, (2) Resolve the action, (3) Describe and apply the results.
- **One decision = one die roll.** Never stretch a single action across multiple rolls just because it takes longer in-game. "You haven't found herbs yet, keep looking?" is press-X-to-continue, not a real choice.
- **Time is an illusion. Game time doubly so.** An action is measured in decisions, not in hours or days. Sweep away dull time with a sentence of narration.
- **Longer actions don't need more complications.** Don't punish players for having a plan that takes a few days by throwing extra random encounters at them. The complexity of the action hasn't changed.
- **An Arbitrary Game Turn ends** when the action is resolved, when something changes, or when the players intervene.
- **Always know what the rest of the party is doing.** When one player acts, ask the others. Evaluate all actions, resolve all actions, then narrate all results.
- **Not everything needs to be a scene.** A failed persuasion check can be narrated in one sentence instead of played out as a full encounter. Think like a movie editor—show what's tense and interesting, cut the rest.
- **Good pacing = controlling the flow of time.** Decide how much detail to show on-screen. Wave away hours or days of non-action with one line of narration when nothing interesting happens.

## Summary

The Arbitrary Game Turn is Angry's framework for resolving any player action, regardless of scale. Whether a character searches a room (10 minutes) or forages in the woods (all day), the resolution process is identical: Evaluate (clarify intent, assess risk, inform the player), Resolve (roll dice, do GM math behind the screen), and Narrate (describe results, sweep away elapsed time). One decision always equals one resolution step—never stretch a single action across multiple rolls just because it takes longer in fiction. The turn ends when the action completes, something changes, or players intervene. When party members split up, evaluate and resolve all actions, then narrate results in sequence. The core insight is pacing: GMs can and should wave away hours or days of uneventful time with a single sentence, and not every interaction needs to be a full scene.

## Actionable Steps

1. **When a player declares an action, run the three-step cycle:** Evaluate → Resolve → Narrate. Don't skip steps or add extra ones.
2. **Resist the urge to add rolls for longer actions.** Foraging all day? One roll. Traveling for a week? One roll (plus any actual encounters). The action's complexity hasn't changed.
3. **During Evaluate, inform the player of costs and risks.** "It'll take several hours and the deep woods might be dangerous. Are you okay with that?"
4. **During Evaluate, ask the rest of the party what they're doing.** Resolve everyone's actions, then narrate all results together.
5. **Resolve everything behind the screen before narrating.** Know the full outcome—including complications—before you start describing what happened. It's easier to adjust a resolved action than to improvise a partial one.
6. **Use cinematic pacing.** If an interaction would be one boring exchange, narrate the result in a sentence. Save full scenes for tense, uncertain, interesting moments.
7. **When an action spans a long time, set a natural breakpoint** ("you want to be back before dark?") and narrate to that point. If players want to abort early, adjust the already-resolved outcome.

> [!tip] Bottom Line
> Every player action—whether it takes six seconds or six days—resolves the same way: Evaluate, Resolve, Narrate. One decision, one resolution. Stop stretching simple actions across multiple rolls and wasting table time. Control the flow of fictional time like a film editor: show the interesting parts, cut the rest.
