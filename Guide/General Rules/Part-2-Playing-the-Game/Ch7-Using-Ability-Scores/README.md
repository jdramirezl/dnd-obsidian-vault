# Chapter 7: Using Ability Scores

The six abilities — Strength, Dexterity, Constitution, Intelligence, Wisdom, and Charisma — define every creature's physical and mental characteristics. This chapter covers how ability scores translate into modifiers, checks, and saving throws.

## Quick Reference: Ability Score Modifiers

| Score | Modifier | Score | Modifier |
|-------|----------|-------|----------|
| 1     | −5       | 16–17 | +3       |
| 2–3   | −4       | 18–19 | +4       |
| 4–5   | −3       | 20–21 | +5       |
| 6–7   | −2       | 22–23 | +6       |
| 8–9   | −1       | 24–25 | +7       |
| 10–11 | +0       | 26–27 | +8       |
| 12–13 | +1       | 28–29 | +9       |
| 14–15 | +2       | 30    | +10      |

> **Formula:** Modifier = floor((Score − 10) / 2)
