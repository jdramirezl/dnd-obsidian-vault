## Quick Reference: Combat Round Sequence

1. **Determine Surprise** — DM decides if anyone is surprised
2. **Roll Initiative** — Everyone rolls; DM ranks turn order
3. **Take Turns** — Each combatant acts in initiative order:
   - **Movement** — Move up to your speed (can split around actions)
   - **Action** — Attack, Cast a Spell, Dash, Dodge, Help, Hide, Ready, etc.
   - **Bonus Action** — If a feature grants one
   - **Free Interaction** — Draw a weapon, open a door, etc.
4. **Reactions** — Triggered between turns (e.g., Opportunity Attacks)
5. **Next Round** — Repeat from step 3 until combat ends
