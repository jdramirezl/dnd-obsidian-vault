# 🔇 Deafened

> [!warning] Deafened
> **Auto-fail hearing checks**
> No verbal component awareness

## Effect

- A deafened creature can't hear and automatically fails any ability check that requires hearing.

## Quick Reference

- **Saves**: Auto-fail hearing checks
- **Combat**: No verbal component awareness

## Common Sources

- [[Blindness Deafness|Blindness/Deafness]]; removed by [[Lesser Restoration]]
