# Condition Cards

Quick-reference cards for all 15 D&D conditions. Each card shows the effect at a glance.

| Condition | Key Effect |
|-----------|-----------|
| [[Blinded\|🙈 Blinded]] | Auto-fail sight checks |
| [[Charmed\|💕 Charmed]] | Can't attack charmer |
| [[Deafened\|🔇 Deafened]] | Auto-fail hearing checks |
| [[Exhaustion\|😩 Exhaustion]] | 6 cumulative levels |
| [[Frightened\|😨 Frightened]] | Disadvantage while source in sight |
| [[Grappled\|🤼 Grappled]] | Speed becomes 0 |
| [[Incapacitated\|💫 Incapacitated]] | Can't take actions or reactions |
| [[Invisible\|👻 Invisible]] | Attacks against have disadvantage |
| [[Paralyzed\|⚡ Paralyzed]] | Auto-fail STR/DEX saves |
| [[Petrified\|🪨 Petrified]] | Turned to stone, weight ×10 |
| [[Poisoned\|☠️ Poisoned]] | Disadvantage on attacks |
| [[Prone\|🔽 Prone]] | Melee attacks against have advantage |
| [[Restrained\|⛓️ Restrained]] | Speed 0, DEX saves disadvantage |
| [[Stunned\|💥 Stunned]] | Auto-fail STR/DEX saves |
| [[Unconscious\|💤 Unconscious]] | Drop everything, prone, auto-fail STR/DEX |
