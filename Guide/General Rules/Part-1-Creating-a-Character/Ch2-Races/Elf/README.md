# Elf

> Elves are a magical people of otherworldly grace, living in the world but not entirely part of it.

## Quick Reference

| Trait | Value |
|---|---|
| **Ability Score** | Dexterity +2 |
| **Size** | Medium (under 5 ft to over 6 ft) |
| **Speed** | 30 ft |
| **Darkvision** | 60 ft |
| **Lifespan** | 750+ years |
| **Languages** | Common, Elvish |
| **Key Traits** | Keen Senses, Fey Ancestry, Trance |
| **Subraces** | High Elf, Wood Elf, Dark Elf (Drow) |
