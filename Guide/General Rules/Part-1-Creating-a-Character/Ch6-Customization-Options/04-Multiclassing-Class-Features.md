## Original Text

### Class Features

When you gain a new level in a class, you get its features for that level. A few features, however, have additional rules when you're multiclassing: Channel Divinity, Extra Attack, Unarmored Defense, and Spellcasting.

#### Channel Divinity

If you already have the Channel Divinity feature and gain a level in a class that also grants the feature, you gain the Channel Divinity effects granted by that class, but getting the feature again doesn't give you an additional use of it. You gain additional uses only when you reach a class level that explicitly grants them to you. For example, if you are a cleric 6/paladin 4, you can use Channel Divinity twice between rests because you are high enough level in the cleric class to have more uses. Whenever you use the feature, you can choose any of the Channel Divinity effects available to you from your two classes.

#### Extra Attack

If you gain the Extra Attack class feature from more than one class, the features don't add together. You can't make more than two attacks with this feature unless it says you do (as the fighter's version of Extra Attack does). Similarly, the warlock's eldritch invocation Thirsting Blade doesn't give you additional attacks if you also have Extra Attack.

#### Unarmored Defense

If you already have the Unarmored Defense feature, you can't gain it again from another class.

#### Spellcasting Focus

Each spell you know and prepare is associated with one of your classes, and you use the spellcasting ability of that class when you cast the spell. Similarly, a spellcasting focus, such as a holy symbol, can be used only for the spells from the class associated with that focus.

## Condensed Version

**Key Concept: Most class features stack normally when multiclassing, but four specific features have special interaction rules that prevent doubling up.**

### Channel Divinity

- You gain the **new Channel Divinity options** from the second class
- You do **NOT** gain extra uses — uses are based on individual class level
- You can choose **any** of your available Channel Divinity effects when you use it
- *Example: Cleric 6/Paladin 4 = 2 uses (from Cleric 6), can pick from either class's options*

### Extra Attack

- **Does NOT stack** across classes — you still only get 2 attacks total
- Exception: Fighter's improved Extra Attack (3 or 4 attacks) works as written
- Warlock's **Thirsting Blade** invocation also doesn't stack with Extra Attack
- ⚠️ *Common trap: Don't take 5 levels in two martial classes expecting 3 attacks*

### Unarmored Defense

- If you already have it from one class, you **cannot gain it again** from another
- You must **pick one** — you use whichever version you got first
- *Applies to Barbarian (CON-based) and Monk (WIS-based) multiclasses*

### Spellcasting Focus

- A focus (holy symbol, arcane focus, druidic focus, etc.) can only be used for spells **from the class associated with that focus**
- A holy symbol does NOT work for your wizard spells, even if you're a Cleric/Wizard
- You may need **multiple foci** if multiclassing between caster classes
