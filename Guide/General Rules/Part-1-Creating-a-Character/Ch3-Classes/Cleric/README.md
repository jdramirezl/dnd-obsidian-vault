## Class Summary

**Hit Die**: d8
**Primary Ability**: Wisdom
**Saves**: Wisdom, Charisma
**Armor**: Light, medium, shields (heavy with some domains)
**Weapons**: Simple
**Core Mechanic**: [[08-Divine-Domain|Divine Domain]] — chosen at 1st level, grants bonus spells, Channel Divinity options, and domain features
