## Quick Reference

| Attribute | Value |
|-----------|-------|
| **Hit Die** | d6 (lowest in the game) |
| **Primary Ability** | Intelligence |
| **Saving Throws** | Intelligence, Wisdom |
| **Armor Proficiencies** | None |
| **Weapon Proficiencies** | Daggers, darts, slings, quarterstaffs, light crossbows |
| **Spellcasting** | Full caster (Intelligence), prepare from spellbook daily |
| **Subclass Name** | Arcane Tradition (chosen at 2nd level) |
| **Subclass Options** | Evocation (Basic Rules); Abjuration, Conjuration, Divination, Enchantment, Illusion, Necromancy, Transmutation (PHB) |

### Key Features by Level

| Level | Feature |
|-------|---------|
| 1st | Spellcasting, Arcane Recovery |
| 2nd | Arcane Tradition |
| 5th | 3rd-level spells (Fireball, Counterspell) |
| 9th | 5th-level spells |
| 17th | 9th-level spells (Wish) |
| 18th | Spell Mastery (at-will 1st/2nd level spells) |
| 20th | Signature Spells (free 3rd-level casts) |

**Tradition Feature Levels**: 2, 6, 10, 14
**ASI Levels**: 4, 8, 12, 16, 19 (5 total)
**Spellbook Growth**: +2 free spells per level, plus any copied from scrolls/books
