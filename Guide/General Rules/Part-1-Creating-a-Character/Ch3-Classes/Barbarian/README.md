## Class Summary

**Hit Die**: d12 (highest in the game)
**Primary Ability**: Strength
**Saves**: Strength, Constitution
**Armor**: Light, medium, shields (no heavy)
**Weapons**: Simple, martial
**Core Mechanic**: [[06-Rage|Rage]] — bonus action for damage bonus, damage resistance, STR advantage
