## What Makes Warlock Unique

- **Short-rest caster**: 1–4 spell slots that recharge on short rest (not long rest)
- **All slots same level**: Every spell is always cast at maximum power
- **Eldritch Invocations**: At-will abilities that provide reliable power between spell slots
- **Two identity choices**: Patron (1st level) + Pact Boon (3rd level) create highly customizable builds
- **Eldritch blast + Agonizing Blast**: The best cantrip + invocation combo in the game
- **Mystic Arcanum**: High-level spells (6th–9th) at 1/long rest each, separate from Pact Magic
