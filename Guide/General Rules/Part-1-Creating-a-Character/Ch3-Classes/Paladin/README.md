## Key Identity

- **Role:** Holy warrior, divine half-caster, party protector
- **Primary Ability:** Strength (melee) + Charisma (spells & auras)
- **Hit Die:** d10
- **Casting:** Half-caster (CHA), prepared spells, spell slots start at 2nd level
- **Subclass (Sacred Oath):** Chosen at 3rd level — Devotion, Ancients, or Vengeance
