> *No one chooses sorcery; the power chooses the sorcerer. Sorcerers carry a magical birthright — an innate power fueled by exotic bloodlines, otherworldly influence, or unknown cosmic forces.*

## Quick Reference

| Attribute | Value |

| **Primary Ability** | Charisma |
| **Saves** | Constitution, Charisma |
| **Armor** | None (Draconic: 13 + DEX natural) |
| **Weapons** | Daggers, darts, slings, quarterstaffs, light crossbows |
| **Skills** | Choose 2 from Arcana, Deception, Insight, Intimidation, Persuasion, Religion |

| **Ritual Casting** | ❌ No |
