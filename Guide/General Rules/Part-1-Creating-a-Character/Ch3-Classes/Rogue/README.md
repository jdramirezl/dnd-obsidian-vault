# Rogue - Index

This directory contains a complete breakdown of the Rogue class — masters of skill, stealth, and precision who exploit enemy weaknesses with devastating Sneak Attacks and unmatched expertise.

## Files in Order

1. [[01-Rogue-Overview|Rogue Overview]] - Class identity, skill and precision, a shady living
2. [[02-Creating-a-Rogue|Creating a Rogue]] - Backstory questions: relationship to the law, triggers
3. [[03-Quick-Build|Quick Build]] - Fast setup: DEX first, Criminal background
4. [[04-Hit-Points-and-Proficiencies|Hit Points and Proficiencies]] - d8 HD, light armor, finesse weapons, 4 skill picks
   - [[04b-Rogue-Level-Table|Rogue Level Table]] - Full 1–20 progression with Sneak Attack scaling
5. [[05-Equipment|Equipment]] - Starting gear choices and recommended setups
6. [[06-Expertise|Expertise]] - Double proficiency bonus on 2 skills (4 total by 6th level)
7. [[07-Sneak-Attack|Sneak Attack]] - Core damage: 1d6–10d6 extra on one attack per turn
8. [[08-Thieves-Cant|Thieves' Cant]] - Secret language: coded speech and criminal symbols
9. [[09-Cunning-Action|Cunning Action]] - Bonus action Dash, Disengage, or Hide every turn
10. [[10-Roguish-Archetype|Roguish Archetype]] - Subclass choice at 3rd level (Thief in Basic Rules)
11. [[11-Ability-Score-Improvement|Ability Score Improvement]] - 6 ASIs at levels 4, 8, 10, 12, 16, 19
12. [[12-Uncanny-Dodge|Uncanny Dodge]] - Reaction to halve one attack's damage (5th level)
13. [[13-Evasion|Evasion]] - No damage on DEX save success, half on fail (7th level)
14. [[14-Reliable-Talent|Reliable Talent]] - Minimum roll of 10 on proficient checks (11th level)
15. [[15-Blindsense|Blindsense]] - Detect hidden/invisible creatures within 10 ft (14th level)
16. [[16-Slippery-Mind|Slippery Mind]] - Wisdom save proficiency (15th level)
17. [[17-Elusive|Elusive]] - No attack can have advantage against you (18th level)
18. [[18-Stroke-of-Luck|Stroke of Luck]] - Turn a miss into a hit or failed check into a 20 (20th level)
19. [[19-Thief-Archetype|Thief Archetype]] - Fast Hands, Second-Story Work, Supreme Sneak, Use Magic Device, Thief's Reflexes
20. *(reserved)*
21. [[21-Assassin|Assassin]] - Bonus proficiencies, Assassinate, Infiltration Expertise, Impostor, Death Strike
22. [[22-Arcane-Trickster|Arcane Trickster]] - Spellcasting (INT), Mage Hand Legerdemain, Magical Ambush, Versatile Trickster, Spell Thief

## Quick Reference

| Attribute | Value |
|-----------|-------|
| **Hit Die** | d8 |
| **Primary Ability** | Dexterity |
| **Saving Throws** | Dexterity, Intelligence |
| **Armor Proficiencies** | Light armor |
| **Weapon Proficiencies** | Simple weapons, hand crossbows, longswords, rapiers, shortswords |
| **Tool Proficiencies** | Thieves' tools |
| **Spellcasting** | None (Arcane Trickster subclass adds it) |
| **Subclass Name** | Roguish Archetype (chosen at 3rd level) |
| **Subclass Options** | Thief (Basic Rules), Assassin, Arcane Trickster (PHB) |
| **Skill Proficiencies** | 4 (most of any class) |

### Key Features by Level

| Level | Feature |
|-------|---------|
| 1st | Expertise (×2), Sneak Attack (1d6), Thieves' Cant |
| 2nd | Cunning Action |
| 3rd | Roguish Archetype |
| 5th | Uncanny Dodge |
| 6th | Expertise (×2 more) |
| 7th | Evasion |
| 11th | Reliable Talent |
| 15th | Slippery Mind (Wisdom save proficiency) |
| 18th | Elusive |
| 20th | Stroke of Luck |

**Sneak Attack**: +1d6 every odd level (1d6 at 1st → 10d6 at 19th)
**Archetype Feature Levels**: 3, 9, 13, 17
**ASI Levels**: 4, 8, 10, 12, 16, 19 (6 total)
